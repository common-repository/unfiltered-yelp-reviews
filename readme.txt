This plugin helps draw interaction to your filtered reviews on yelp, which could possibly help your SEO.

== Description ==

This plugin helps draw attention and interaction to your filtered reviews on yelp, which won't get them out of the filter, but it will allow you to outrank their webpage for your company with your own for your business wordpress blog.

== Installation ==

1. Upload unFiltered to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place Yelp Review Widget in a Widget Area
4. Copy paste the necessary data to create each review
